AI Chat Interaction 도움말

이곳에 AI 채팅 상호작용 기능에 대한 자세한 사용법을 작성합니다냥.


📖 [1] AI 대화 활성화하기

Gemini API 또는 로컬 Ollama를 사용하여 AI와 대화할 준비를 합니다.

  • 🔑 Gemini API 사용하기: Google AI Studio에서 API 키를 발급받아 환경설정에 입력
  • 💻 Ollama 사용하기: 윈도우에 Ollama 설치 후 로컬 모델 설정

👤 [2] 나를 캐릭터에게 소개하기

캐릭터가 나를 더 잘 알 수 있도록 프로필과 자기소개를 작성합니다.

  • 🆔 프로필 메시지: 유저 상태에 따라 자동 표시
  • ✏️ 자기소개 작성: 성격, 취향, 말투 등 자유롭게 작성

👥 [3] 캐릭터 프리셋 추가하기

대화할 캐릭터를 생성하고, 여러 캐릭터를 상황에 맞게 관리합니다.

  • ➕ 프리셋 생성: 친구 추가 버튼 클릭
  • ⚙️ 설정 진입: 톱니바퀴 클릭 → 캐릭터 설정 열기
  • 🗃️ 프리셋 관리: 여러 캐릭터를 상황별로 설정 가능

🎨 [4] 캐릭터 만들기

이름, 성격, 배경, 말투 등 캐릭터의 상세한 설정을 만듭니다.

  • 🏷️ 기본 정보: 이름, 성별, 성격
  • 📊 지능/친밀도: 캐릭터 능력 수치 설정
  • 📖 성격과 배경: 가치관, 말투, 외형 등 서술
  • 💬 대사 예시: 말투를 반영한 문장 입력

▶️ [5] 대화 시작하기

원하는 캐릭터를 선택하고, 상태 설정을 통해 대화를 시작합니다.

  • 🖱️ 프리셋 클릭: 메인 창에서 캐릭터 선택
  • 🔔 유저 상태 설정:
    • 💡 활성화: 캐릭터가 먼저 메시지를 보냄
    • 🚫 방해금지: 메시지는 오지만 알림 없음
    • 💤 비활성화: 메시지도 오지 않음

⭐ 알아두면 좋은 기능

AI 화면 인식, 프리셋 공유 등 특별한 추가 기능을 활용해보세요.

  • 👀 AI 화면 인식: 화면 상태를 캐릭터가 인식해 대화 가능 (주의: 개인정보)
  • 📤 프리셋 공유: 캐릭터 설정을 파일로 주고받을 수 있음 (VRM 미포함)
