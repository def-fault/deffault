🔹 1. 캐릭터 AI 대화 기능 활성화하기

AI 캐릭터와 대화하려면 먼저 AI 기능을 활성화해야 해요.

두 가지 방식 중 하나를 선택해서 사용하세요:

  • ✅ 방법 1: Gemini API 사용하기 (클라우드 기반)
  • Google Cloud 또는 Google AI Studio에서 Gemini API 키를 발급받습니다.
  • 앱의 톱니바퀴(⚙️) 아이콘을 클릭하여 환경설정에 들어갑니다.
  • API 키를 입력하고 ‘적용’ 버튼을 클릭하면 설정 완료!

  • ✅ 방법 2: 로컬 AI 모델 사용하기
  • Desktop Maid 설치 시 Gemma 3가 같이 설치 되어 바로 사용하실 수 있습니다.
  • 환경설정에서 API 사용 또는 로컬 사용을 선택할 수 있습니다.
